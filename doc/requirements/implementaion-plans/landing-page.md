# 🚀 Landing Page Implementation Prompt – Otaku Community Project

## 1. Project Overview

You are implementing a **Landing Page** to introduce the **Otaku Community Project** – a social platform for anime, manga, and otaku culture lovers.

The landing page must:

- Clearly explain **what the project is**
- Highlight **key features & value**
- Encourage users to **Sign Up / Log In**
- Have a **modern, anime-inspired, clean UI**

---

## 2. Target Audience

- Anime / Manga / Otaku fans
- Age: 16–35
- Users familiar with social platforms (Facebook, Twitter, Reddit)
- Prefer visual, dynamic, and community-driven experiences

---

## 3. Page Goals

- First impression of the product
- Communicate core value in **5 seconds**
- Convert visitors into **registered users**
- Build trust & excitement about the community

---

## 4. Layout Structure (Top → Bottom)

### 4.1 Hero Section

- Large headline (project slogan)
- Short description (1–2 sentences)
- Primary CTA button: **"Join the Community"**
- Secondary CTA: **"Explore Features"**
- Background: anime-style gradient / illustration

**Example content:**

- Headline: _"Where Otaku Connect & Create Together"_
- Subtext: _"Share posts, follow creators, discuss topics, and build your anime universe."_

---

### 4.2 Key Features Section

Display **4–6 feature cards** with icon + title + short description.

**Required features:**

- 📝 Create & Share Posts
- ❤️ Like / Comment / Vote
- 🔔 Real-time Notifications
- 👥 Follow Users & Topics
- 🖼️ Media-rich Content (Images, Videos)
- 🧵 Personalized Feed

---

### 4.3 How It Works

3-step simple flow:

1. Sign up / Log in
2. Follow topics & users you like
3. Interact, post, and grow your profile

Use numbers or illustrations.

---

### 4.4 Community Value Section

Explain **why this platform is different**:

- Built for otaku, not generic social media
- Topic-based discussions
- Clean feed algorithm
- Safe & moderated environment

---

### 4.5 Preview / Screenshot Section

- Show mockups or placeholders of:
  - Home Feed
  - Profile Page
  - Post Detail
  - Notification UI

(Static images or placeholder cards if real screenshots are not available)

---

### 4.6 Call To Action (CTA)

- Strong message encouraging sign-up
- Button: **"Start Your Otaku Journey"**

---

### 4.7 Footer

Include:

- Project name & short description
- Links: About, Features, Contact
- Social links (placeholder)
- Copyright

---

## 5. Design Requirements

- Style: Modern, Anime-inspired, Clean
- Color palette: Dark mode friendly (purple / orange / pink accents)
- Typography: Sans-serif, readable
- Responsive (Desktop / Tablet / Mobile)
- Subtle animations (hover, fade-in, scroll)

---

## 6. Technical Requirements

- Use **React + Tailwind CSS**
- Component-based structure
- Clean, readable code
- No backend integration required (mock data allowed)
- Buttons can be non-functional placeholders

---

## 7. Output Expectation

- Full Landing Page UI
- Reusable components
- Well-structured JSX
- Easy to extend later

---

## 8. Tone & Copywriting

- Friendly
- Energetic
- Community-driven
- Otaku-culture aware (but not cringe)

---

## 9. Do NOT

- Overload text
- Use generic corporate wording
- Add unnecessary animations
