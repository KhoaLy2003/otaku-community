# Product Vision

## Problem

The otaku community needs a dedicated space to connect, share, and discuss their interests without the noise of general social media. Existing platforms are either too broad (Facebook/Twitter) or too fragmented (forums/Reddit).

## Target Users

- **Primary Users:** Japan enthusiasts, anime/manga fans, Japanese language learners, travelers.
- **Secondary Users:** Content creators, Community moderators.

## Core Value

**Key Benefits:**

- **Topic-Based Organization:** Enable community-driven content discovery through focused topics (like Reddit).
- **Social Timeline:** Provide personalized feeds based on user interests and follows (like Twitter).
- **Rich Content Support:** optimized for anime, manga, JLPT learning, travel, food, and culture.

## Non-Goals

- Direct messaging between users (MVP)
- Video content support
- Advanced JLPT learning modules
- Event management system
- Gamification features (badges, XP, missions)
- Mobile native applications
- Real-time chat functionality
- Third-party API integrations (MyAnimeList, AniList) - _Note: Backend integration planned but frontend features minimal in MVP_

## Success Criteria

**Metrics:**

- User engagement rate (likes, comments per post)
- Growth in active topics
- User retention rate

**Milestones:**

- [ ] MVP Launch (Core social features)
- [ ] Topic discovery improvements
- [ ] Integration with external anime/manga databases
